# imaple-crawler
A BBS crawler for imaple.tw.

*Note: `uao_decode.py` in this project has been modified to support `Python2` and `Python3`.*

## Usage
- Example: `./crawl.py nthu.course 13080 13090`

    ```
    ./crawl.py [board_name] [start_postid] [end_postid]
    ```


## Requirements
- Python3
- pyte
